# 第01章 向量引论
## 1.1 向量与线性组合
### 标量与向量的区别
- **标量**：只有大小，没有方向。例如，温度、质量、时间、能量。
- **向量**：既有大小又有方向。例如，速度、力、加速度。

向量的大小：向量在空间中的长度
方向：所指的方向

向量的起点通常是原点 (0, 0)。
