### 回到上一级目录
```
cd ../
```

### 回到根目录
```
cd /
```

### 回到家目录
```
cd ~
或者：
cd
```

### 显示你现在的位置
```
pwd
```

### 在当前位置新建文件夹 
```
mkdir 文件夹名
```

### 在指定目录位置，创建文件夹，并创建父文件夹 
```
mkdir -p /a/b/文件夹名
```

### 在当前目录下新建文件
```
touch 文件名
```

### 删除文件
```
rm 文件
```

### 删除文件夹
```
rm -r 文件夹
```

### 强制删除不询问
```
rm -rf 文件
```

### 移动源文件到目标文件夹中
```
mv 文件  文件夹
```

### 修改文件A的名字为文件B
```
mv 文件A 文件B
```

### 查看文件中的全部信息(适合查看小文档)
```
cat 文件名
```

### 文件名查找
```
find 搜索路径 -name "文件名关键词"

例子 
find / -name "passwd" find / -name "ifcfg-*"
```

### 文件内容查找

```
语法
grep -参数 要查找的目录范围 
参数
-n 显示查找结果所在行号 
-R 递归查找目录下的所有文件
例子 grep aries /etc grep aries /etc/passwd
```
