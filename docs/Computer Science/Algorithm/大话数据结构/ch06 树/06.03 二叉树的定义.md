由一个根结点和两颗互不相交、分别称为根节点的左子树和右子树的二叉树组成。或者结点为 0 的空二叉树

![](images/Pasted%20image%2020241111193424.png)

二叉树的特点
- 每个结点有≤两颗子树，
- 左右子树有顺序，只有一颗子树的时候也要区分，这是不同形态

二叉树的五种基本形态
- 空二叉树
- 只有一个根结点
- 根结点只有左 / 右子树➡
- 根结点左右子树都有

![](images/Pasted%20image%2020241111193820.png)

特殊二叉树
- 斜树，如树 2 和 5
- 满二叉树：
	- 左右子树都存在；
	- 非叶子结点的度（也就是子树的数量）一定是 2
	- ![](images/Pasted%20image%2020241111193931.png)
- 完全二叉树 
	- 一颗二叉树自上而下，从左到右编号，这个编号如果按顺序排好且没有缺失时，则称完全二叉树
	- ![](images/Pasted%20image%2020241111194647.png)

二叉树的性质
- 二叉树的第 $i$ 层上最多右 $2^{i-1}$ 个结点 $(i\ge 1)$ （某一层的结点）
- 深度为 $k$ 的二叉树最多有 $2^{k}-1$ 个结点 $(k\ge 1)$（整棵树的结点）
